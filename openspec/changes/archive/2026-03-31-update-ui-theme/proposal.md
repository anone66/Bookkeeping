# 变更：优化UI主题与视觉特效

## 为什么
目前的界面虽然现代，但缺乏深浅色（白天/黑夜）模式切换，且可以进一步提升视觉质感。用户需要更简约高效、具有优美光效等高级特效的炫酷现代化界面，同时需要优化弹窗和提示样式，使整体体验更加统一、沉浸。

## 变更内容
- **主题切换**：为应用增加浅色（白天）和深色（黑夜）主题平滑切换能力。
- **界面优化**：优化登录页和记账页的布局与卡片样式，使其简约高效美观。
- **高级视觉特效**：引入高级光效等炫酷特性增强视觉体验（如动态渐变背景、流光发光边框等），并做到深层结合。
- **组件优化**：优化弹窗（Toast）及提示框的样式与动画，使其更符合现代设计语言。

## 影响
- 受影响规范：`personal-ledger`（修改界面需求），`ui-theme`（新增主题与特效需求）
- 受影响代码：`static/style.css`, `static/index.html`, `static/app.js`
